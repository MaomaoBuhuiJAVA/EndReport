# 龙湾区国科温州第二幼儿园智慧成长平台项目报告

## 一、项目定位

本项目将原机器人控制台升级为面向“龙湾区国科温州第二幼儿园”的官方网站式智慧成长平台，覆盖园所概览、功能室展示、成长照片、园所资料库、AI 资料问答、用户登录注册和云宝机器人管理。

页面整体采用教育官网风格，结合真实幼儿园照片、毛玻璃导航、渐变虚化首屏、功能室树形入口和照片放大预览，让公开访问更像正式园所官网，而管理员登录后可进入资料维护和云宝管理能力。

## 二、已完成成果

1. 权限与账号系统
   - 增加登录、注册、退出、修改密码功能。
   - 数据库真实写入用户数据。
   - 普通用户只查看园所信息、功能室、公开照片、云宝简介。
   - 管理员才能查看云宝设备状态、监控画面、运行日志和 WASD 控制。
   - 首页和 `/api/overview` 已按用户角色返回不同数据，避免普通用户拿到设备敏感信息。

2. 官方网站式 UI 优化
   - 首页背景照片下移，并在顶部加入渐变虚化。
   - 顶部导航改为 iOS 毛玻璃效果，登录入口放在左上角。
   - 搜索功能移到顶部。
   - 登录注册页面重新设计为独立用户中心。
   - 页面背景加入层次网格、柔和色块和教育官网风格配色。

3. 功能室展示
   - `room` 文件夹功能室照片已处理并写入数据库。
   - 功能室入口从词云改为“树”结构，按钮作为树叶漂浮分布。
   - 不同功能室按钮带有对应图标，例如建构、科学、阅读、未来工作坊等。
   - 功能室详情支持照片展示与放大预览。

4. 园所资料库
   - `ziliao` 文件夹资料已解析并写入 Neon 数据库。
   - 已入库资料共 16 份，生成 AI 检索分块 89 个。
   - 获奖汇总表 `.xlsx/.et` 已读取表格内容入库。
   - 2022 学年教师荣誉 `.doc` 已作为荣誉资料入库。
   - 基本情况、省二资料、社团、纲要、指南、智慧校园发言稿、教职工名单等已入库。
   - 体积过大的教师用书类资料已按要求跳过，不占用资料库。
   - 园所资料卡片支持点击查看数据库正文。
   - 管理员可在页面中新增资料，资料会写入数据库并生成 AI 检索分块。

5. AI 问答与照片联动
   - AI 对话优先检索数据库资料。
   - 增强中文关键词、同义词和 n-gram 检索。
   - 用户询问“功能室有没有照片”“有没有图片”“参观环境”等问题时，会返回相关照片卡片。
   - 没有模型 key 或模型调用失败时，也会用数据库检索结果生成兜底回答。

6. 云宝机器人模块
   - 删除心跳展示入口，保留后端心跳接口用于设备状态更新。
   - 移动控制改为 WASD。
   - 管理员可查看设备状态、电量、班级空间、监控画面和日志。
   - 普通用户只能查看云宝简介和公开展示画面。

## 五、仍可继续完善的不足

1. 旧 Word `.doc` 的解析能力有限
   - 已使用 `word-extractor` 和二进制文本兜底提取。
   - 个别旧 WPS/Word 文件正文提取质量仍依赖文件内部格式。

2. 真实机器人硬件协议尚未接入
   - 当前 WASD 指令会写入数据库命令表。
   - 后续可接 MQTT、WebSocket 或设备 HTTP 网关，让真实机器人消费指令。

3. 照片后台管理还可增强
   - 当前照片来自本地素材处理后入库。
   - 后续可增加管理员上传、分类、审核、删除和排序功能。

4. 资料详情可继续做表格化展示
   - 目前 Excel 表格已转为文本入库。
   - 后续可为获奖表增加真正的表格视图、筛选和导出。
